# Lavanda Stainless Steel Kitchen Website

A modern, responsive business website for Lavanda Stainless Steel Kitchen, located in Kozhikode, Kerala.

## Features
- Premium minimal design
- Fully responsive (Mobile & Desktop)
- Interactive Project Gallery with Lightbox
- Scroll Reveal animations
- Dynamic Navigation with scroll effects
- One-click contact via WhatsApp and Phone

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to View
This website is hosted for free using GitHub Pages.
